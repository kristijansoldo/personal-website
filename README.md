Personal website
------------
Iterations of my personal website: www.kristijansoldo.com

Built with
---------------
- GatsbyJS
- ReactJS

Requirements
---------------
- NPM ^v6.9.0
- Node ^v10
- Gatsby CLI v2.12.15

Development
---------------
```bash
$ git clone https://github.com/kristijansoldo/personal-website.git
$ cd personal-website
$ npm install
$ gatsby develop
```
